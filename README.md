## Coca-Cola Stock Movement Forecasting Using Markov Chains
Developed discrete-time Markov chain forecasting models to analyze and predict stock movement behavior across multiple Coca-Cola-related equities using historical market data from 2002-2024. Built transition probability matrices, conducted stationarity testing, and evaluated forecasting performance using stochastic modeling techniques.

Core Concepts:  
Time Series Analysis • Statistical Forecasting • Financial Modeling • Markov Chains • Data Validation
